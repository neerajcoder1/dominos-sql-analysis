# Domino’s Pizza Sales Analysis
![Pizza-Toppings-Banner](https://github.com/user-attachments/assets/1dc38a81-0fdf-4ad6-88a4-98b8b38c53ad)

## Overview
This project analyzes Domino’s pizza sales data using SQL, Python, and Plotly.  
The goal is to extract business insights such as revenue trends, customer behavior, and product performance.

---

## Dataset
The dataset contains 5 tables:

- `orders` → order-level data  
- `order_details` → items in each order  
- `pizzas` → price and size  
- `pizza_types` → category and name  
- `customers` → customer details  

---

## Tools Used

- SQL (SQLite)
- Python (Pandas)
- Plotly (Visualization)
- Jupyter Notebook

---

## Project Workflow

1. Data Loading (CSV → SQL Database)  
2. Data Cleaning (handling missing values)  
3. SQL Analysis (joins, aggregation, filtering)  
4. Visualization (Plotly charts)  
5. Business Insights  

---

## Key Analysis

### 1. Orders Trend
- Orders analyzed by hour and date  
- Peak demand identified  

### 2. Revenue Analysis
- Total revenue calculated  
- Category-wise contribution  

### 3. Product Performance
- Top-selling pizzas  
- Most popular sizes  

### 4. Customer Analysis
- Top spending customers  
- Repeat customer behavior
